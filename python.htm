pythonを使おう

pythonは使いますか。

pythonとはプログラミング言語の一種です。

難易度も低いです。

とりあえずやってみましょう。

pythonのインストール

まずこのページにアクセスします。

そしたら、上の画像の赤い四角で囲まれている場所をクリックします。

ダウンロードが終わったら、上の画像の赤い四角で囲まれている場所をダブルクリックします。

すると、このような画面がでます。上の画像の赤い四角で囲まれている場所をクリックします。

すると、このような画面になります。上の画像のようにチェックを入れて,赤い四角で囲まれている場所をクリックします。

すると、このような画面になります。上の画像のようにチェックを入れます。

チェックのついていないところは、使うときににチェックを入れましょう。

しばらくすると、このような画面になります。これでpythonが使えるようになりました。

vscodeにpythonを対応させよう。

まだ、vscodeをインストールしていない人はこのページを参考にインストールしましょう。

vscodeを起動します。

vscodeが起動したら、赤い四角で囲まれている場所をクリックします。

そしたら、左上の四角に「python」と打ち込んで、赤色の四角があるすべての場所をクリックします。

「python」はpythonをvscodeに対応させる拡張機能です。

「python indent」はpythonのプログラムをきれいに字下げするための拡張機能です。

「python Extended」は「python」を拡張するための拡張機能です。

しばらくして、このようになったらvscodeでpythonが使えるようになります。

使ってみよう

早速pythonを使ってみましょう。

デスクトップにpythonというフォルダを作ります。(フォルダの名前は何でもいいです。)

次に、「ファイル」をクリックして「フォルダを開く」をクリックします。

作成したフォルダを選択して「OK」をクリックします。

そしたら、上の画像の赤い四角で囲まれている場所をクリックします。

そしたら、index.pyと打ち込んでエンターキーを押します。ファイルの名前は何でもいいですが、必ず最後の文字に「.py」をつけてください。

vscodeにこのように入力します。

print( "HelloWorld")

そしたら「ctrl」キーと「s」キーを同時に押します。

このようにすることで、ファイルを保存することができます。

実行してみよう

windowsの場合はコマンドプロンプトを、macの場合はターミナルを開きましょう。

そして次のように入力してエンターキーを押します。

cd desktop

これでデスクトップに移動しました。今度はpython(作成したフォルダ)に移動します。

次のように入力てエンターキーを押します。

cd python

これで、python(作成したフォルダ)に移動しました。

次に実行します。

次のように打ってエンターキーを押します。index.pyのところにはファイルの名前を入れます。

python index.py

次にこのように表示されれば成功です。

HelloWorld

このように、コマンドプロンプトまたはターミナルから、「cd python」(pythonはフォルダ名)と入力することで、で実行したいファイルがあるフォルダに移動して

「python index.py」(index.pyはファイル名)と入力することでpythonファイルを実行できます。

日付を表示してみよう。

プログラム(index.py)をこのように変更します。

import datetime

time = datetime.date.today()

print(time)

ファイルを保存しましょう

実行してみましょう。現在の年と現在の月と現在の日が表示されたら成功です。

このプログラムの意味を説明します

これだけでは面白くないので,日付を表示させてみましょう。

1行目の「import datetime」はpythonで標準で入っているライブラリ「detatime」を読み込んでいます。

「detatime」とは、時間を扱えるライブラリです。

2行目の「time = datetime.date.today()」は「time」という名前の変数を作って、現在の年と現在の月と現在の日を入れています。

「変数」とはデーターを入れる箱のようなものです。変数に入っているものを「値」といいます。

ここではでてきませんが、その箱に入るデーターの種類のことを「型」といいます。

pythonでは「変数名=入れたいもの」と書くことで、型に関係なく変数の値を変えられます。

3行目の「print(time)」は変数「time」の値を読みだして、表示しています。
